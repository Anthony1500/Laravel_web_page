Antivirus

Los programas antivirus y el software de protección de computadoras
tienen como objetivo examinar datos como páginas web, archivos, software y aplicaciones en busca de malware y actuar rápidamente para eliminarlos.

La mayoría de ellos ofrecen protección en tiempo real para prevenir
amenazas entrantes, además realizan escaneos periódicos
del dispositivo para detectar posibles amenazas y se actualización conforme se descubren nuevas amenazas.

Un ejemplo de antivirus es: Malwarebytes Anti-Malware

VPN

Tecnología de red que permite establecer una conexión segura y cifrada entre dos dispositivos a través de Internet. Esta tecnología crea un túnel encriptado que protege la comunicación entre los dispositivos y evita que terceros puedan verla.

La VPN es muy útil cuando se accede a Internet desde una red pública, ya que protege el tráfico de Internet de posibles espías en la red y garantiza la privacidad y seguridad de la información transmitida.

Un ejemplo de VPN es: NordVPN

Firewall

Un firewall o cortafuegos es una medida de seguridad informática que se utiliza para controlar el tráfico de red entre diferentes dispositivos y redes, actuando como un filtro que bloquea el tráfico no autorizado o malicioso, mientras permite el tráfico seguro y autorizado.

El firewall puede ser implementado a nivel de software o de hardware y se utiliza para proteger la red de una organización, así como los dispositivos y datos de los usuarios contra posibles ataques y amenazas de seguridad.

Administrador de contraseñas

Se emplea un software para almacenar y gestionar de manera segura las contraseñas de múltiples cuentas de usuario.

El software se encarga de generar contraseñas seguras y complejas, las cuales son difíciles de adivinar o crackear, y las almacena en una base de datos encriptada protegida por una contraseña maestra.

Un ejemplo de Administrador de contraseñas es: Bitwarden

Herramientas de análisis forense

Son aplicaciones diseñadas para recopilar y examinar datos digitales
con el objetivo de identificar y recolectar evidencia digital en investigaciones criminales o legales.

Estas herramientas se utilizan para analizar el contenido de dispositivos electrónicos, como computadoras, teléfonos móviles, unidades de almacenamiento, entre otros, y permiten recuperar información eliminada, identificar el origen de un ataque informático o realizar un seguimiento de la actividad en línea

Un ejemplo de una herramienta es: Autopsy
